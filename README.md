Pygame Framework
================

Pygame Framework is a library that helps making games with Pygame.
It provides useful tricks such as:

* State machine
* Consistent game loop
* Object-oriented syntax

## Requirements

* [Python][1] 2.7
* [Pygame][2] 1.9
